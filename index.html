Questions fréquentes

L'application est-elle vraiment gratuite ?

Oui, PenaltyDz est entièrement gratuit. Aucun abonnement ni achat in-app n'est requis pour utiliser toutes les fonctionnalités.

Dans quelles villes est disponible PenaltyDz ?

PenaltyDz est disponible dans toutes les grandes villes d'Algérie et s'étend progressivement à d'autres régions.

L'application est-elle disponible sur iOS ?

Pour le moment, PenaltyDz est uniquement disponible sur Android. Une version iOS est prévue pour le futur.
